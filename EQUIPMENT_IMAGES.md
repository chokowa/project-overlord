# 装備アイテム画像の追加方法

## 対象アイテム

以下の4つの装備アイテムに画像を設定できます：

### パーツスロット
- **大容量コンデンサ** (`high_capacity.png`)
- **精密スタビライザー** (`stabilizer.png`)

### ギアスロット
- **気合ハチマキ** (`headband.png`)
- **フカPay** (`fuka_pay.png`)

## 画像の配置方法

1. 画像ファイルを準備（推奨: 64x64～128x128ピクセル、PNG形式、透過背景）
2. `assets/`フォルダに以下のファイル名で配置：
   ```
   GAME/assets/
   ├── high_capacity.png    （大容量コンデンサ）
   ├── stabilizer.png       （精密スタビライザー）
   ├── headband.png         （気合ハチマキ）
   └── fuka_pay.png         （フカPay）
   ```
3. ゲームをリロード（F5キー）

## 注意事項

- ファイル名は厳密に一致させてください
- 画像がない場合は自動的に絵文字アイコンで表示されます
- 正方形の画像を推奨（歪み防止）
